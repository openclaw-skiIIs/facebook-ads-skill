# Facebook Ads Skill for Claude Code

A Claude Code skill for generating high-converting Facebook ad campaigns using proven copywriting frameworks.

## Features

- Generate 3 ad variants per campaign (AIDA, PAS, BAB frameworks)
- Audience targeting suggestions with interest-based segments
- Hook writing for video and image ads
- A/B testing structure ready to launch
- Saves session to a local markdown file for reference

## Install

```bash
claude plugins install github:AmitAvital1/facebook-ads-skill
```

## Usage

Once installed, ask Claude:

> "Create a Facebook ad for my product"

Or invoke directly:

```
/facebook-ads
```

## Frameworks Included

| Framework | Best For |
|-----------|----------|
| AIDA | Cold audiences, brand awareness |
| PAS | Pain-point driven products |
| BAB | Transformation products (fitness, SaaS, education) |
